Classic's share grammar: expressing the range mechanic

Today's Classic share text leads with a hints-used bar (🟩⬜) and buries the range mechanic — the thing a Classic player is actually doing every turn — in a plain-text stats line. This lab holds 8 structurally distinct grammars for the share body, each built to foreground range width and contained vs. missed instead of hint count. Every option is shown against the same two scenarios so they're comparable apples-to-apples.

Spoiler-safety floor (non-negotiable across every option): no option below ever prints an absolute calendar year — only relative widths, distances, and directions. That's the same anti-leakage property the current implementation already holds; none of these options regress it.
Scenarios held constant across all 8 options — Win: guesses narrowed 250y → 60y → 15y, contained the target, scored 85/100. Loss: guesses narrowed 250y → 90y → 40y, missed by 5 years (too early), scored 0/100. Puzzle #347 in both. Header/footer family (Chrondle #347 … https://chrondle.app) is fixed by the shared module and out of scope for this lab — only the body grammar varies.
